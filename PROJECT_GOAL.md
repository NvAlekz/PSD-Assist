# Pokemon Showdown Assistant

## Objetivo General

Crear un asistente inteligente para Pokemon Showdown capaz de:

- Leer automáticamente el estado de una batalla.
- Detectar información relevante del combate.
- Mantener una representación interna del estado actual.
- Analizar opciones disponibles.
- Recomendar las mejores jugadas.
- Mostrar sugerencias mediante una interfaz visual.
- Registrar partidas para análisis posterior.

## Requisitos

- Código limpio y modular.
- Tests automatizados.
- Documentación.
- Arquitectura extensible.

## Objetivo MVP

La primera versión debe:

1. Detectar Pokemon activos.
2. Detectar HP.
3. Detectar estados alterados.
4. Detectar movimientos disponibles.
5. Recomendar una jugada básica.
6. Mostrar recomendaciones en una interfaz simple.
